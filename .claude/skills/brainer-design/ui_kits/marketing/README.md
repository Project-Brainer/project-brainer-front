# Marketing UI Kit — Brainer

Single-page landing site demonstrating the marketing surface for Project-Brainer.

## Sections included

1. **Nav** — logo, links, sign-in + CTA.
2. **Hero** — eyebrow pill, 88px serif headline with italic emphasis on "don't document it.", lede, two CTAs, big product preview frame with mini-canvas.
3. **Logo row** — social-proof strip (placeholder company names).
4. **Features** — "Five node types, infinite systems" — 3×2 grid of primitive cards.
5. **Simulation split** — copy + code block showing generated prompt.
6. **Quote block** — 44px italic pull quote, avatar + attribution.
7. **CTA** — dark full-width card.
8. **Footer** — copyright + links.

## Files

- `index.html` — all markup.
- `styles.css` — all layout, type, cards, product frame.

No JSX used; marketing is a static page and didn't need React.

## Notes

- Headlines use **Instrument Serif** with italic emphasis — the brand's signature move.
- The product preview uses the actual **mini-node** style from the web-app kit, reinforcing brand consistency.
- Kept to a single page — content is illustrative, not exhaustive. Real copy and real logos from you will replace the placeholders.
